# json_from_csv
Este programa, dada la ubicación de un archivo CSV, lee su contenido y guarda un nuevo archivo con el mismo nombre pero en formato JSON. 

La conversión se realiza mediante dos clases principales: Estudiante y ConversorCSVtoJSON.

La clase Estudiante representa los datos de un estudiante, incluyendo su ID, nombre y apellido. 
La clase ConversorCSVtoJSON se encarga de leer el archivo CSV, crear objetos Estudiante a partir de los datos leídos y escribir esos datos en un archivo JSON.

La ubicación del archivo CSV se proporciona como entrada al programa, que luego crea un nuevo archivo JSON con el mismo nombre que el archivo CSV, pero con la extensión .json.

Este programa es útil para convertir archivos CSV de datos de estudiantes u otras entidades en formato JSON de una manera fácil y automatizada.
